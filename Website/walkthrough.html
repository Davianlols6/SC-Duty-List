Navigation

You can navigate the app by switching between "Home" and "Settings" in the navigator bar on the bottom part of your screen.

Home Page

On the home page, you will see 2 input fields. The input field on the left is called the "Date input" while the input field on the right is called the "Mode selector".

By default, on start up, the Date input should show the current date while the Mode selector should show "Overview"

Home: Date input

The Date input allows you to select the date for which you want to view the duty list data for.

For example, if you want to view the duty list data for 29 November 2021, simply click on the Date input and change the date to 29 November 2021. On success, you should be able to see the title "Duty list for Monday, 29 November 2021" on the top of the Home screen.

Home: Mode selector

The Mode selector has 2 modes which you can change between. The first and default mode is the Overview mode which will show you an Overview duty list for the date you select.

The second mode is called the "Personalised Mode". To select to this mode, simply select your name. Personalised Mode will show only the duties which are assigned to the name you select.

Settings Page

To go to the Settings Page, simply click on the settings button in the navigator bar.

The Settings Page allows you to customise some aspects of the websites and much more.

Settings: Theme

In the Theme setting, you can select any one of 3 options.

The first and default option is "Use Device Settings". If you select this option, the app will switch between light mode and dark mode according to your device setting.

The second option is "Always use Light Mode". If you select this option, the app will always use light mode.

The third and final option is "Always use Dark Mode". If you select this option, the app will always use dark mode.

Settings: Update

In the Update page, you can check for updates or force an update. These two options will not work in offline mode

You can check for updates by pressing on the "Check for updates" button. By default, the app will automatically check for updates every 5 mins.

You can force an update to happen by pressing on the "Force Update" button. Do note that you may have to press this button twice before seeing any change.
